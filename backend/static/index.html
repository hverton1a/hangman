The words are scraped from the web and stored in a dockerized Mysql Server DB.

Designed to be multi-language, storing original and sanitarized form, only dependant to be alphabetical.

Although only word in portuguese are available at the moment.

The Container whare deployed at a GCP - Google Cloud - VPS instance, with docker-compose.

To Handle Tsl and serve HTTPS/WSS for secure connetions Whas span up a NGINX Docker container as reverse proxy with a Certbot Docker container to handle with LetsEncript the certificates.

The workflow was dockerized, coded remotely inside the container.

Features:

  • Modular Word Scrapper

    get the word from dictionary site or api implemented with recipes, being flexible and ease to aquire word from another sources, just add the recipe and the scrape will deal with it.

  • dockerized

    Speeding up the deployment and scale

  • Designed to be Alphabet dependant, but not language dependant

    The Word Table was designed to receive sanitarized and original word forms, and the language was designed one-to-many relationship.

It´s still in development, at the moment it´s not possivel to create or log in a user, also the game statistics was not implemented.
